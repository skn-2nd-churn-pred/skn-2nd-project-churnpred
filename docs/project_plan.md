## 3. 검증 계획
- 분리: 고객1행→stratified / 반복기록→group / 시계열→시간순 · (Test 20%, Val 20%)
- 전처리기는 **Train에만 fit**, Val/Test엔 transform만
- 모델·임계값은 **Validation에서** 선택, **Test는 한 번만**
- 금지: Test로 모델/임계값 선택, 누수 컬럼 사용, 전체 전처리 후 split
